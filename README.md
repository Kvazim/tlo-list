Тестовое для НПО "Интеллектуальные Технические Системы".

Тестовое ReactJs
Разработать одностраничное приложение на React, выложить в свой репозиторий и
прислать ссылку.
1. Приложение должно отображать светофорные объекты в виде:
списка;
объектов на карте.
2. Требования к списку:
для каждого объекта отображается по крайней мере его номер и адрес;
в зависимости от статуса (поле mode) объекты в списке должны отличаться;
возможность фильтрации списка по номеру/адресу;
возможность фильтрации списка по статусу объекта (поле mode).
3. Требования к картографическому отображению:
использовать leaflet/react-leaflet;
в зависимости от статуса (поле mode) объекты на карте должны отличаться;
при клике на объект отображается тултип с информацией о выбранном объекте;
4. Требования по взаимодействию списка и карты:
при фильтрации объектов в списке, объекты на карте также фильтруются;
при клике на объекте в списке на карте происходит позиционирование на объекте.
